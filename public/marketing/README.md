# Marketing Materials Directory

This directory contains downloadable marketing materials for affiliates to use when promoting DoorIQ.

## Directory Structure

- `social/` - Social media graphics (Instagram, Facebook, LinkedIn)
- `banners/` - Website banner graphics
- `email/` - Email templates (HTML)
- `logos/` - Logo assets (PNG, SVG)
- `templates/` - Text templates for posts and emails

## Adding Materials

When adding new marketing materials:
1. Place files in the appropriate subdirectory
2. Update `app/affiliate/materials/page.tsx` to include the new material
3. Ensure files are optimized for web (compressed images, minified HTML)

## File Naming Convention

- Use lowercase with hyphens: `instagram-post.png`
- Include size in filename if multiple sizes: `banner-1920x600.png`
- Use descriptive names: `email-introduction-template.html`
